# 90-Day Rollout Plan

## Days 1–14

- Confirm central kitchen agreement framework
- Collect supplier compliance docs
- Define first 10–12 SKUs
- Build SKU cost and pricing table
- Confirm selected halal-certified meal rule
- Start site search

## Days 15–30

- Shortlist 3–5 City sites
- Check use class and lease restrictions
- Check power capacity
- Estimate occupancy cost
- Prepare Food Business Registration file
- Prepare food safety system outline

## Days 31–60

- Negotiate lease
- Confirm electrical capacity with commercial electrician
- Submit Food Business Registration
- Finalise store layout
- Finalise equipment list
- Build inventory rules
- Build label templates

## Days 61–90

- Fit-out and equipment install
- POS, kiosk and stock system setup
- Staff recruitment and training
- Full menu reheating test
- Cold chain delivery test
- Peak-hour simulation
- Soft opening
- SKU and process adjustment

# Site Selection Checklist

## Target

- Area: City of London / London financial district
- Size: 80–100 sqm
- Use: Class E / café / food retail compatible
- Power: three-phase, target 60–75kVA
- Model: front-of-house approx. 70%, back-of-house approx. 30%

## Priority

1. Close to large office clusters
2. Close to tube / commuter flow
3. Rent and total occupancy cost reasonable
4. Shopfront visibility

## Questions for landlord / agent

| Question | Why it matters |
|---|---|
| What is the current use class? | Must support café / food retail |
| Is light reheating permitted? | Core operating model |
| Is eat-in seating permitted? | Store positioning |
| Is takeaway ancillary permitted? | Grab-and-go sales |
| What is the current power capacity? | Equipment load |
| Is it three-phase power? | Essential |
| Can power be upgraded? | Future growth |
| What is the rent? | Unit economics |
| What is the rateable value? | Business rates |
| What is service charge? | True occupancy cost |
| Are signs allowed? | Brand visibility |
| Are deliveries allowed in the morning? | Central kitchen supply |
| Is waste storage available? | Daily operations |
| Is there any restriction on vending machines? | Smart vending setup |

## Red flags

- No three-phase power
- Landlord refuses light food use
- High service charge
- No delivery access
- No waste handling route
- Rent above model tolerance
- Lease prohibits signage, seating or food reheating

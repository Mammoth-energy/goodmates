# Compliance & Licensing

## Regulatory strategy

GOODMATES Grab & Go should be positioned as:

**Grab-and-go café / Asian lunch store / eat-in with ancillary takeaway.**

Avoid describing the business as a hot food takeaway shop.

## Key compliance areas

| Area | Requirement | Action |
|---|---|---|
| Food Business Registration | Required before trading | Submit to local council at least 28 days before food operations |
| Food safety system | HACCP / SFBB required | Prepare food safety file before opening |
| Allergens | Must be clear and accurate | Build allergen matrix for all SKUs |
| Labels | Required for prepacked / PPDS-style products | Product name, ingredients, allergens, batch, use-by, storage, reheating |
| Alcohol | Not sold | No alcohol licence needed under current plan |
| Late-night refreshment | Not relevant if closing by 20:00 | Avoid trading hot food/drink 23:00–05:00 |
| Planning use | Prefer Class E | Confirm café / grab-and-go / light reheating permitted |
| VAT | Conservative model assumes 20% | Accountant to confirm final treatment |
| Halal | Only selected certified meals | Label selected certified meals only |
| Vending machine | Food safety + temperature | Include in HACCP and temperature records |

## Store compliance records

- Supplier approval file
- Delivery temperature log
- Cold storage temperature log
- Vending machine temperature log
- Reheating parameter sheet
- Cleaning checklist
- Daily stock and use-by date log
- Allergen matrix
- Recall procedure
- Complaint log

# Risk Control Table

| Risk | Priority | Control |
|---|---|---|
| Supplier contract not in place | P0 | Request standard agreement and onboarding pack |
| Supplier compliance documents incomplete | P0 | Collect BRC, HACCP, insurance, allergen, recall docs |
| SKU margin too low | P0 | Build cost sheet before launch |
| Site power insufficient | P0 | Require three-phase 60–75kVA target |
| Use class unsuitable | P0 | Confirm Class E / café / food retail compatibility |
| Food registration delayed | P0 | Submit at least 28 days before operations |
| Allergen labelling error | P0 | Maintain allergen matrix and double check labels |
| Halal claim inaccurate | P0 | Only label certified SKUs |
| Peak-hour service too slow | P1 | Use cold grab-and-go, kiosk, app, vending and pickup split |
| Vending machine fault | P1 | Start with one in-store unit only |
| Food waste high | P1 | Inventory alert at 10 and 5 remaining; no-waste plan |
| Drink queue slows food | P1 | Keep drinks machine-made and separate from pickup flow |
| Rent too high | P0 | Set strict occupancy cost ceiling |

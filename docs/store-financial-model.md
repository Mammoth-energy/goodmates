# Store Financial Model｜Summary

All prices include VAT. Conservative planning assumes 20% VAT.

## Core assumptions

| Item | Base case |
|---|---:|
| Main meals per day | 550 |
| Monthly trading days | 22 |
| Average main meal price incl. VAT | £8.60 |
| Average main meal net revenue ex. VAT | £7.17 |
| Average meal cost | £3.50 |
| Drink attach rate | 25% |
| Average drink price incl. VAT | £4.20 |
| Average drink cost | £0.80 |
| Target launch budget | £200,000 |

## Monthly model

| Item | Base case |
|---|---:|
| Monthly main meals | 12,100 |
| Monthly drinks | 3,025 |
| Gross sales incl. VAT | approx. £116,765 |
| Net revenue ex. VAT | approx. £97,304 |
| Direct food and drink cost | approx. £44,770 |
| Gross profit | approx. £52,534 |
| Estimated fixed operating cost | approx. £37,500 |
| Monthly EBITDA | approx. £15,000 |

## Payback

| Scenario | Payback |
|---|---:|
| Conservative | Not expansion-ready |
| Base case | 12–18 months |
| Strong case | 7–9 months |

## Operating target

- Survival line: 400 meals/day
- Healthy line: 500–550 meals/day
- Expansion line: 600+ meals/day
